Personal Library

  1. POST a title to /api/books to add a book, and a JSON object with the title and a unique _id will be returned.
  2. GET /api/books to retrieve an aray of all books containing title, _id, & commentcount.
  3. GET /api/books/{_id} to retrieve a single object of a book containing title, _id, & an array of comments (empty array if no comments present).
  4. POST a comment to /api/books/{_id} to add a comment, and the book's object will be returned (similar to get /api/books/{_id}).
  5. DELETE /api/books/{_id} to delete a book from the collection. If successful, a 'delete successful' message will be returned.
  6. If a request contains a book that doesn't exist, then a "no book exists" message will be returned.
  7. Send a DELETE request to /api/books to delete all books in the database. If successful, a 'complete delete successful' message will be returned.
